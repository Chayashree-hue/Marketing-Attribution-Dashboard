# 📊Portfolio Project- Marketing Attribution Performance Dashboard

## 🔍 Overview
This Power BI dashboard analyzes marketing performance using multiple attribution models including First Click, Last Click, Linear, and Time Decay.

---

## 📈 Key Features
- KPI Cards (Conversions, Revenue, CTR)
- First Click Attribution Analysis
- Last Click Attribution Analysis
- Linear Attribution Model
- Time Decay Attribution Model
- Campaign Performance Breakdown
